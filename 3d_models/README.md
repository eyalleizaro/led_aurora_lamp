# 3D Models

Place the lamp body, diffuser, holder, or other 3D model files in this folder.

Suggested file types:

- `.stl`
- `.obj`
- `.step`

Recommended names:

```text
aurora_lamp_body.stl
diffuser.stl
motor_mount.stl
electronics_holder.stl
```

If the design stays only in Tinkercad, add exported STL/OBJ files here when they are ready.
